macOS · Local · Private

Transcribe meetings.
No cloud. No subscription.

trnscrb lives in your menu bar, auto-detects meetings, transcribes locally with Whisper, and makes every transcript searchable from Claude Desktop.

Recommended install
# Install
brew tap ajayrmk/tap
brew install trnscrb
trnscrb install

# Launch
trnscrb start

How it works

1
Meeting detected

Mic active for 5 s + meeting app running — Google Meet, Zoom, Slack Huddle, Teams, FaceTime.

2
Recording

Audio captured via mic or BlackHole virtual driver for system + mic combined.

3
Transcription

Whisper small model runs entirely on-device using Apple Silicon Metal — no internet required.

4
Speaker labels

pyannote.audio assigns SPEAKER_00, SPEAKER_01 etc. Run trnscrb enrich to replace them with inferred names.

5
Saved to Claude Desktop

Plain .txt in ~/meeting-notes/, instantly searchable from Claude Desktop via MCP.

Install

Recommended

Homebrew

Isolated, managed, easy to update or remove.

brew tap ajayrmk/tap
brew install trnscrb
trnscrb install
pip

pip / uv

For Python developers who prefer pip or uv.

# pip
pip install trnscrb && trnscrb install

# uv
uv tool install trnscrb && trnscrb install

trnscrb install handles BlackHole audio driver, HuggingFace token for diarization, Whisper model download (~500 MB, one-time), Claude Desktop MCP config, and launch-at-login.

Claude Desktop integration

After trnscrb install, Claude Desktop gains these tools automatically:

Tool Description
start_recordingStart capturing audio
stop_recordingStop and transcribe in the background
recording_statusCheck if recording or transcribing
get_last_transcriptFetch the most recent transcript
list_transcriptsList all saved meetings
get_transcriptRead a specific transcript
get_calendar_contextCurrent or upcoming calendar event
enrich_transcriptAdd summary + action items via Claude API

CLI reference

CommandDescription
trnscrb startLaunch the menu bar app
trnscrb installGuided setup / re-check dependencies
trnscrb listList saved transcripts
trnscrb show <id>Print a transcript
trnscrb enrich <id>Add summary + action items (needs ANTHROPIC_API_KEY)
trnscrb watchHeadless auto-transcribe, no menu bar
trnscrb mic-statusLive mic activity monitor — useful for debugging
trnscrb devicesList audio input devices

Privacy

Everything runs on your machine. No audio or transcripts leave your device unless you explicitly run trnscrb enrich, which sends transcript text to the Claude API for summarisation.
